About This Project

The Emotion Web App is designed to detect human facial emotions using a Convolutional Neural Network (CNN) model trained on the CK+ dataset. Users can upload images of faces, and the app predicts the corresponding emotion from five categories: Happy, Sad, Neutral, Angry, and Surprise. This project demonstrates practical applications of deep learning in emotion recognition, human-computer interaction, and AI-based user experience improvements.
