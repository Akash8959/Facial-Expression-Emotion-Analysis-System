# Facial Expression Emotion Analysis System

# Project Overview
The Facial Expression Emotion Analysis System is a deep learningâ€“based application designed to detect and classify human emotions from facial expressions. The system analyzes both static images and real-time webcam video streams to identify emotions such as Happy, Sad, Angry, Neutral, and Surprise.
It leverages Convolutional Neural Networks (CNN) along with computer vision techniques to provide accurate and real-time emotion recognition through a simple web-based interface.

# Objectives
To automatically detect human emotions from facial expressions
To implement a CNN-based deep learning model for emotion classification
To support real-time emotion detection using a webcam
To provide a user-friendly web interface for interaction and visualization.

# Technologies Used
Programming Language: Python
Deep Learning: TensorFlow, Keras
Computer Vision: OpenCV
Web Framework: Flask
Frontend: HTML, CSS, Bootstrap
Database: MySQL / CSV (for logs and records)

# Features
Emotion detection from uploaded images
Real-time facial emotion recognition using webcam
CNN-based deep learning model for high accuracy
Interactive and responsive web dashboard
Displays prediction results and model performance metrics

# Installation & Setup
Step 1: Install Dependencies
        pip install -r requirements.txt
Step 2: Run the Application
        python app.py
Step 3: Open in Browser
        http://localhost:5000

# Dataset
FER-2013

# Model Description
The emotion recognition model is built using a Convolutional Neural Network (CNN) architecture. It consists of multiple convolutional layers for feature extraction, pooling layers for dimensionality reduction, and dense layers for emotion classification. The model is trained and evaluated using accuracy and loss metrics.
